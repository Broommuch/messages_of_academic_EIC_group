# messages_of_academic_EIC_group



- 背景：本项目为华中科技大学电子信息与通信学院“**电信学院学术交流群**”（群号778969928）聊天记录的备份文件。

  由于此群被迫关闭入群通道且面临解散风险，特别保存其群文件与珍贵学生交流聊天记录，其中部分聊天文件/群文件因太大或未及时保存而**失效**。

- 基本信息：目前备份群聊文件时间：2024/08/29~2026/05/18	

​	本项目使用工具：https://github.com/shuakami/qq-chat-exporter.git 

- 当前目录下`group_20240829_20260518.html` 和`/resources`保存了html版聊天文件和相关群文件，点击html文件即用。

  而`/group_20240829_20260518_chunked_jsonl`为json版，可独立使用。
